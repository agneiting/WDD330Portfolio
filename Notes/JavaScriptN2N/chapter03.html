CHAPTER THREE NOTES - ARRAYS, LOGIC, & LOOPS

Arrays Create an array literal: const arrayName = []; const arrayName= ['first item', 'second item', 'third item']; Add value to array: arrayName[0] = 'first item'; Remove value from array: delete arrayName[0]; See array: arrayName; Pop - Remove last item from array: arrayName.pop(); Shift - Remove first item from array: arrayName.shift(); Push - Add new value to end of array: arrayName.push('new item'); Unshift - Add new value to beginning of array: arrayName.unshift('new item'); When you sort an array of numbers, it will sort it alphabetically, so you have to use a callback to sort numerically. Sets Like an array but cannot include any duplicate values. Memory Leak: Occurs when a program retains references to values that can no longer be accessed in its memory. This means that memory is being used to store values that are no longer required by the program, effectively wasting system resources. This causes to program to run more slowly. Maps Similar to objects but unlike objects, • they can use any data type as a key. • You can find the number of key-value pairs by using the "Size" property. • Maps focus solely on the storage and retrieval of key-value pairs. • You can't access the value directly, and must use the get() method. Chapter Summary Arrays are an ordered list of values Multidimensional arrays are arrays that contain other arrays Arrays have lots of methods that can be used to manipulate items in the array Sets are new in ES6 and are ordered lists of non-duplicate values Maps are new in ES6 and are ordered lists of key-value pairs We can use an if and else statement to control the flow of code The switch statement can be used instead of multiple if and else statements A while loop and do ... while loop can be used to repeat a block of code while a condition is still true A for loop works in a similar way to a while loop, but has a different syntax A for-of loop can be used to iterate over an array Sets and maps are enumerable, so can also be looped over using a for-of loop Questions Destructuring array's is a little fuzzy.
